\# Task 5 - End-to-End Object Detection Application



\## Overview

This project demonstrates an end-to-end computer vision application that performs automatic object detection on visual data using a trained deep learning model. The system is capable of identifying and localizing objects in unseen images and videos.



\---



\## Features

\- Object detection on test images

\- Video-level inference with bounding box visualization

\- Frame-wise prediction and reconstruction into video format

\- Optional enhancement with audio integration for presentation



\---



\## Workflow

1\. Trained a pre-trained YOLO model on a custom dataset

2\. Generated optimized weights after training

3\. Applied the trained model on unseen test images and videos

4\. Produced annotated outputs with bounding boxes

5\. Converted processed frames into final output video



\---



\## Output

\- Annotated images with detected objects

\- Processed video showing real-time detection results

\- Final visualization-ready output for demonstration purposes



\---



\## Results

The model successfully detects and localizes trained object classes in unseen data with good accuracy, demonstrating practical applicability of deep learning-based object detection.



\---



\## Conclusion

This project showcases how modern vision models can be adapted to build real-world applications for automated object detection in images and videos.
