This task involves preparing a dataset for object detection using a real-world video. A video was selected and frames were extracted from it using FFmpeg. From the extracted frames, relevant images were chosen and organized into train, validation, and test folders based on different time intervals to ensure proper dataset distribution.



The dataset is structured into images/train, images/val, and images/test folders. A data.yaml file was created to define the dataset paths and class information for future YOLO training.



This step mainly focuses on dataset preparation, cleaning, and organization so that it can be used later for annotation and model training. No labeling is included in this task.



The dataset is now ready for the next stage of annotation and training in the YOLO pipeline.
